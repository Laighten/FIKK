#### 一、如何理解HTML语义化？

​	HTML前端语义化：就比如我们在写html代码的时候，该用div标签的时候用div标签，该用h标签的时候用h标签，该使用li标签的时候用li标签；不要全用div标签去嵌套实现。HTML语义化的好处就是：

- 让人更容易读懂（增加代码的可读性）
- 让搜索引擎更容易读懂

#### 二、哪些是块级元素哪些是内联元素？

块级元素(display:block/table)：

- div
- h1
- h2
- table
- ul
- ol
- p

内联元素(display:inline/inline-block):

- span 
- img
- input
- button
